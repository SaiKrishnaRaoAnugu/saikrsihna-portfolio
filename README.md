# 🚀 Sai Krishna Rao Anugu — Portfolio Website

A modern, production-ready portfolio website built with **Next.js 16**, **TypeScript**, and **Tailwind CSS**.  
Designed with a clean, dark, AI-focused aesthetic and interactive particle background.

Live Website: https://saikrishna.vercel.app  
GitHub: https://github.com/SaiKrishnaRaoAnugu/saikrsihna-portfolio

---

## ✨ Features

- ⚡ Built with Next.js (App Router)
- 🎨 Modern dark-themed UI
- 🌌 Interactive animated particle background
- 📱 Fully responsive design
- 🚀 Optimized for performance & SEO
- ☁️ Deployed on Vercel
- 🧠 AI-focused professional branding

---

## 🛠 Tech Stack

- **Framework:** Next.js 16
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Animations:** tsParticles
- **Deployment:** Vercel
- **Version Control:** Git & GitHub
